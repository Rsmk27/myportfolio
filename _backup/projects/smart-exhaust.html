Smart Exhaust System

An intelligent safety system that detects hazardous gas leaks and automatically triggers active ventilation.

Arduino Uno C++ (Embedded) MQ-2 Gas Sensor Relay Logic
Live Demo GitHub Repo

01. The Problem

In domestic kitchens and industrial environments, gas leaks (LPG, smoke, carbon monoxide) pose silent but deadly risks. Passive ventilation is often insufficient to clear a room quickly during a leak, and manual intervention requires a human to enter a hazardous zone. There is a need for a reactive, autonomous system that mitigates the danger the moment it is detected.

02. The Solution

The Smart Exhaust System acts as an active guardian. Using a sensitive MQ-2 gas sensor, it continuously monitors air quality. Upon detecting a threshold breach, it bypasses the need for human action and instantly triggers a high-power exhaust fan via a relay module. It also features a manual override, ensuring the user retains ultimate control over the system.

04. Implementation Details

Analog Sensing

The MQ-2 Sensor provides variable analog voltage output based on gas concentration, which is read by the Arduino's ADC (Analog-to-Digital Converter).

Threshold Logic

Implemented a dynamic threshold in C++. If sensorValue > safeLimit, the system enters ALARM state, activating the relay and status LEDs.

Electrical Isolation

Used an opto-isolated Relay Module to safely control the high-voltage (AC 220V) exhaust fan from the low-voltage (5V) logic circuit.

06. Challenges & Learnings

Challenge: Sensor Calibration

The MQ-2 sensor requires pre-heating and is sensitive to temperature changes, causing false triggers.

Solution: Implemented a "warm-up" delay in the boot code and calibrated the threshold value in fresh air before deployment.

Challenge: Relay Chattering

When gas levels were exactly at the threshold, the fan would rapidly switch on and off.

Solution: Added software hysteresis (a gap between the turn-on and turn-off values) to stabilize the control loop.

07. Future Improvements

  • IoT Integration: Add an ESP8266 to send push notifications to a smartphone when a leak is detected.
  • Gas Shutoff Valve: Integrate a solenoid valve to physically cut off the gas supply, not just vent the room.
  • LCD Display: Show real-time PPM (Parts Per Million) levels.
